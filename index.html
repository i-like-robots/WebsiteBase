WebsiteBase

WebsiteBase is more than a boilerplate. It combines ideas from OOCSS and SMACSS to help create a maintainable code base for large scale website development.

Features

  • Documented CSS caveats and code notes
  • Scalable project structure and conventions
  • Guidelines for HTML, CSS and JavaScript coding style
  • Full set of CSS3 LESS mixins
  • Neat typography out of the box
  • Useful layout classes to get stuff done
  • Cross-browser form elements
  • Nestable grid
